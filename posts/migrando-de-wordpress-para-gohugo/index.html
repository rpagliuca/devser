Migrando de Wordpress para Hugo (gohugo)

8 Junho 2019

Hoje migrei a plataforma deste blog de Wordpress para Hugo (https://gohugo.io/), pelos seguintes motivos:

  • Aumento de performance
  • Redução de custos
  • Abertura do código-fonte: https://github.com/rpagliuca/devser

Na verdade Hugo não é uma plataforma, e sim, uma biblioteca capaz de produzir um blog estático, a partir de arquivos declarativos .yml e markdown. Uma outra biblioteca bastante conhecida, similar ao Hugo, é a Jekyll (https://jekyllrb.com/). Como agora o blog ficou estático, estará hospedado na plataforma Github Pages.

Uma pequena desvantagem é que todos os comentários antigos foram perdidos, já que a partir de agora estaremos utilizando o serviço Facebook Comments.
