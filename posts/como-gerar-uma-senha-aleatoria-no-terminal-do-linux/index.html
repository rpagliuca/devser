Como gerar uma senha aleatória no terminal do Linux

18 Junho 2019

Gerar uma senha aleatória no terminal do Linux não é só possível, como altamente recomendado, pois assim você garante que a senha gerada, em nenhum momento, passe pelas mãos de terceiros.

Execute o comando abaixo no terminal:

cat /dev/random | stdbuf -o0 tr -cd '[:graph:]' | stdbuf -o0 head -c 32
